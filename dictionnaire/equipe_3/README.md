# Définition
## Un dictionnaire est une strucure de données qui permet de stocker des données en les identifiant par une clé. Chaque valeur est associée à sa clé et peut être de n'importe quel type. Contrairement aux listes, il faut utiliser des accolades {} pour délimiter. Les dictionnaires sont toujours de type dict

# Opérations
- items(): retourne la paire clé:valeur asoociée à l'indice
- len(): affiche le nombre d'items du dictionnaire. (Une paire = un item)
- dict(): permet de créer un dictionnaire

# Avantages et inconvénients
## Avantages
- La recherche est rapide
- Les dictionnaires sont facilement modifiables

## Inconvénients
- Utilise beaucoup de mémoire

# Exemples d'utilisation
- Profils d'utilisateurs
- Stockage de données
